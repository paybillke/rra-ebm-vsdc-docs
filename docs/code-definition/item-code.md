# Item Code

Each item must have a unique **itemCd**. This field is **mandatory**, and no two items can share the same `itemCd`.

## Structure of `itemCd`

The `itemCd` is generated by combining the country of origin, product type, packaging unit, quantity unit, and a sequential number.

### Example

```text
RW2NTBA0000012
```

### Breakdown

| Segment | Description |
|---------|-------------|
| RW | Country of Origin (Rwanda) |
| 2 | Product Type (Finished Product) |
| NT | Packaging Unit (NET) |
| BA | Quantity Unit (Barrel) |
| 0000012 | Sequential number, incrementing from `0000001` onward |
